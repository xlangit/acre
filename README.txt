Sistema de acreditaciones para el portal de www.maineventproducciones.com. 


El desarrollo esta pautado para realizarlo en 7 días y se otorgara una semana de prueba para cualquier cambio.

El desarrollo constara de la administración de las categorías del personal, diagramación de las acreditaciones
por categoría, vista previa de cada acreditación por usuarios,  impresión de las acreditaciones, 
y historial de impresión por cada acreditación.

Desarrollo del sistema de acreditaciones.
.-	Administración de categorías de personal (fotógrafo, comité organizacional, etc.)
         Agregar, borrar, editar nombres de categorías.
.-	Asignar diagramación de acreditación por categoría de personal.
         Se subirá la imagen de la acreditación y se creara la diagramación de la información
         variable por usuario, como datos personales, empresa, etc.
.-	Listado de usuarios para un evento en especifico
          Se podrá ver una vista previa de su acreditación así como el historial de las impresiones 
          de la misma.
 
Se creara el sistema basado en PHP, JavaScript, AJAX, JQUERY, CSS, HTML5, MYSQL.
Documentación y prueba del portal: 
.-	Se incluirá un manual de usuario y un manual de sistema.

Fases generales
Fase I. Desarrollo interactivo del sistema. Etapa de desarrollo del sistema, 
prueba de las diferentes páginas, diseño de la interfase, pruebas de impresión
y reformulación de los códigos de ser necesario.
Duración: 1 semana.
Fase II. Prueba y cambios. Etapa de prueba del sistema para detectar fallos o realizar cambios, toques finales.
Duración: 1 semana.
